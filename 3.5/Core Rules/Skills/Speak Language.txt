언어 구사(Speak Language) (없음; 숙련자 전용)

행동(Action)
적용되지 않습니다.

재시도(Try Again)
적용되지 않습니다. 언어 구사 체크에는 실패가 없습니다.

언어 구사 스킬은 다른 스킬처럼 작동하지 않습니다. 언어는 다음과 같이 작동합니다.

1레벨에 하나나 두 개의 언어를 알고 시작합니다(종족에 기반합니다). 시작할 때의 지능 보너스마다 추가 언어를 알게 됩니다.

다른 스킬을 배우는 것처럼 언어 구사를 획득할 수 있습니다. 하지만 등급을 얻는 것이 아니라, 말할 수 있는 새로운 언어를 선택합니다.

언어 구사 체크는 수행하지 않습니다. 언어를 알거나 혹은 모르거나 둘 중 하나일 뿐입니다.

문자를 아는 캐릭터(문해력을 갖추도록 스킬 포인트를 투자하지 않은 야만용사를 제외한 모든 캐릭터)는 자신이 말하는 모든 언어를 읽고 쓸 줄 압니다. 각 언어는 문자를 가지고 있고, 때로는 여러 언어가 같은 문자를 공유하기도 합니다.
